# 📚 Bangladesh Elite Publishing Houses Directory

![HTML](https://img.shields.io/badge/HTML-5-orange)
![CSS](https://img.shields.io/badge/CSS-3-blue)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow)
![Status](https://img.shields.io/badge/Status-Active-success)

A clean and lightweight web application that lists the **most elite and reputable book publishing houses in Bangladesh**.

This project provides a structured directory of top publishers including their **address, contact number, website, and publishing focus**. It also highlights the **Top 5 most prestigious publishers** in the country.

The interface is designed to be **minimal, searchable, and responsive**, making it easy for writers, researchers, and readers to quickly explore trusted publishing houses.

---

# ✨ Features

• Directory of elite publishing houses in Bangladesh
• Highlighted **Top 5 prestigious publishers**
• Searchable table interface
• Clickable publisher websites
• Clean and minimal UI
• Fully responsive layout (Desktop + Mobile)
• Lightweight – runs directly in the browser

---

# 🧠 Why This Project Exists

Finding reliable publishing houses in Bangladesh can be difficult for new authors.

This project solves that problem by creating a **simple digital directory** where writers can quickly discover reputable publishers and access their contact information.
